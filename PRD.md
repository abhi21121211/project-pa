## 1. ğŸ“Œ Product Overview

Developers and students build complex projects, but recruiters rarely understand the full functionality. Missing context = missed opportunities.

**AI Project Presenter** is a small plugin (npm/pip package) that developers embed into their project. It adds a **floating icon** in the corner. When a recruiter clicks it, they can:

### 1. Full Project Presentation Mode

A guided tour automatically starts:

- Popup appears
- Mascot animation
- Voice narration
- Automated navigation (auto-click buttons, redirect pages)
- Highlights UI elements
- Explains functionality and tech stack

### 2. Current Page Guide Mode

Explains only the current page's components with hotspots and small tooltips.

The entire presentation script is auto-generated by AI using code analysis + LLM.
